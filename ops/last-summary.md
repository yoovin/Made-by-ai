- 플랫폼 소개, 서비스 추가 4단계 규칙, 현재 서비스 현황, 개발 원칙 섹션 추가
- 허브 카드에서 들어온 사용자가 바로 구조를 이해할 수 있도록 전용 설명 콘텐츠 구성
